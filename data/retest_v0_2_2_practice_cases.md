# TRIM v0.2.2 Practice Cases

These practice cases are not part of the formal retest. Use them to rehearse
primary/context evidence selection, uncertainty calibration, and question-log
entries. No answer key is supplied in the coder-facing package.

## Practice Case 1: Direct Explanation

**Source type:** invented neutral vignette for training only  
**Scope:** local passage only

- `PRACTICE_EXPLAIN_S1`: A town clock stops during a storm.
- `PRACTICE_EXPLAIN_S2`: The keeper says the lightning broke the clock's
  mechanism and points to the scorched gear.
- `PRACTICE_EXPLAIN_S3`: The council delays the scheduled bell-ringing until
  the gear is replaced.

Task:

1. Select one to three primary evidence segment IDs.
2. Add context segment IDs if needed.
3. Decide which segment is most important for your annotation and why.
4. Record any question you ask yourself.

## Practice Case 2: Shared Context Permission

**Source type:** invented neutral vignette for training only  
**Scope:** supplied related cases permitted

- `PRACTICE_SHARED_A_S1`: Witness A says the lantern was lit before midnight.
- `PRACTICE_SHARED_A_S2`: Witness A says the gate was open.
- `PRACTICE_SHARED_B_S1`: Witness B says the lantern was dark after midnight.
- `PRACTICE_SHARED_B_S2`: Witness B says the gate was already locked.

Task:

1. Code one local witness statement first.
2. Then code the same statement with shared context permitted.
3. Note whether your field choices change when the related statements are
   available.
4. Log any uncertainty threshold question.
