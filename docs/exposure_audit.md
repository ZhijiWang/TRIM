# Exposure Audit

The exposure-audit design compares:

- a locked human-pre record;
- a frozen AI-generated record shown in the AI-review condition;
- a later human current response;
- a no-AI second-pass control response where available.

The intended condition difference is the AI panel. The design supports descriptive comparison. It does not by itself establish a causal effect of model exposure.

The current repository includes synthetic dry-run materials and protocol drafts. It does not include approved human-subject data.
