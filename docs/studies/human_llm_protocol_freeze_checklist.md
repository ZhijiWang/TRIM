# Human-LLM Friction-Locus Protocol Freeze Checklist

Status: incomplete. This checklist is a gate for later protocol-freeze,
sample-freeze, researcher-coding, and model-execution tasks. It does not freeze
a sample, authorize coding, or authorize model execution.

## Can be completed before sample definition

- [ ] Design B formally adopted.
- [ ] No external human recruitment permitted under current protocol.
- [ ] Institutional ethics/governance status recorded without claiming
  exemption.
- [ ] Manual freeze requirements specified.
- [ ] Prompt freeze requirements specified.
- [ ] Study-local hashing rules specified.
- [ ] Sample selection log template ready.
- [ ] Exclusion-reason codes confirmed.
- [ ] Actor-specific schemas validated.
- [ ] Human template validated.
- [ ] Model template validated.
- [ ] Run-manifest template validated.
- [ ] Allocation manifest validated.
- [ ] Qualitative case-selection algorithm frozen.
- [ ] Primary and secondary analysis hierarchy frozen.
- [ ] Minimum support rules frozen.
- [ ] Cost ceiling recorded.
- [ ] Stop conditions recorded.
- [ ] Released walkthrough and frozen records verified unchanged.

## Cannot be completed before candidate universe or sample freeze

- [ ] Candidate universe recorded.
- [ ] Candidate pool created without empirical coding.
- [ ] Sample selection log populated.
- [ ] Sample manifest prepared.
- [ ] Sample manifest hash recorded.
- [ ] Source rights confirmed for every case.
- [ ] Source packet specification frozen.
- [ ] Source packet hashes recorded.
- [ ] Textual rights status confirmed.
- [ ] Canonical-language and translation/gloss policy frozen.
- [ ] Context-window and paratext-removal rules frozen.
- [ ] Allocation manifest populated.
- [ ] Allocation manifest hash recorded.
- [ ] Counterfactual tests frozen.
- [ ] Predicted-confusion table frozen against the frozen manual.
- [ ] Prompts A/B/C frozen.
- [ ] Ablation execution order or randomization rule frozen.
- [ ] Model parameters and tool/browsing availability rules frozen.
- [ ] Ethics/privacy status confirmed.

## Cannot be completed before researcher coding

- [ ] Researcher self-record status documented.
- [ ] Human record lock workflow frozen.
- [ ] Researcher records completed.
- [ ] Human record hashes computed and verified.
- [ ] No AI output visible before human lock.
- [ ] No participant data involved.
- [ ] Any future second-human extension requires a new protocol and
  ethics/governance review.
- [ ] No human coding begun before freeze authorization.

## Cannot be completed before model execution

- [ ] Run manifests populated.
- [ ] Raw-output hashes computed immediately after receipt.
- [ ] Parsed-record hashes computed separately from raw-output hashes.
- [ ] Stability-run completion recorded.
- [ ] No model run begun before execution authorization.
- [ ] No empirical case data committed outside the approved manifest and frozen
  source packets.
