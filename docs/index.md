# TRIM-HAA Documentation

TRIM-HAA is the active project in this repository. It provides software for preserving and auditing structured human and model annotation records.

Installed-package functionality covers record validation, lock verification, provenance handling, and comparison. Source-checkout-only demonstrations include the author-only walkthrough and synthetic dry-run workflow.
