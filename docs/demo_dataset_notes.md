# Demonstration Dataset Notes

The ten-case TRIM corpus brings together four *Zuo zhuan* divination cases, three *Macbeth* prophecy cases, and three *In a Grove* testimony cases.

Each annotation records evidence, anchor, function, threshold-rationale signature, rationale, and coder metadata.

The corpus supports four comparisons: shared function with different signatures, shared cue with different functions, shared broad form with different signatures, and contested-threshold review.

These cases establish schema expressivity, workflow traceability, and comparative payoff. Generated tables locate structural relations; researcher-authored readings appear in [`substantive_demo_interpretations.md`](substantive_demo_interpretations.md).

Independent coding and out-of-sample work form the next empirical stages.
