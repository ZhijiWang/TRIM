# Provenance

TRIM-HAA provenance records describe annotation lineage and assistance conditions. They support local audit questions such as:

- who or what produced a record;
- which record was a parent record;
- whether a later human response followed AI exposure or a no-AI second pass;
- which prompt and model-run artifacts were recorded.

The current walkthrough model artifact is locally auditable but not externally reproducible from recorded metadata alone. The exact public model identifier, provider-side configuration, sampling configuration, and full session context are not reconstructable.
