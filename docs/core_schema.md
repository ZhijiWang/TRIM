# Core Schema

The Core record stores a submitted annotation. It includes:

- annotation identity and case identity;
- actor identity and actor type;
- annotation stage;
- selected evidence segment IDs;
- function label;
- rationale mechanism;
- uncertainty;
- rationale note;
- optional alternative pathway fields;
- record status.

The schema records what was submitted. It does not decide whether the annotation is correct.

Current templates remain in `data/trim_haa_core_template.csv` and related `data/trim_haa_*` files.
