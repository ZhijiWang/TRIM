# TRIM-HAA Participant Core Guide

This guide explains the fields you will complete. It uses synthetic examples only.

## Primary Evidence

Select the smallest set of text segments that most directly supports your judgment.

Example: if a synthetic sentence says, "Mira locked the door, then checked the empty hallway twice," you might select only the segment about checking the hallway if your judgment concerns caution.

## Function Label

Choose the project-defined analytic category that best fits the case. Use the label guide provided for the session. If none fits, use the available no-fit option if the project includes one.

## Rationale Mechanism

Choose the project-defined type of connection between the evidence and the function label.

Example: if the evidence shows a character repeating an action, the mechanism might concern repetition, contrast, escalation, confirmation, or another project-defined relation.

## Uncertainty

Use one of:

- Low
- Medium
- High

Uncertainty is not failure. Use higher uncertainty when the evidence is ambiguous, the category boundary is unclear, or another defensible reading remains.

## Rationale Note

Write a short explanation of why the selected evidence supports the chosen label and mechanism.

Example: "The repeated hallway check supports caution because it shows the character actively verifying safety rather than merely noticing the hallway."

## Alternative Pathway

Indicate whether another complete and defensible interpretation remains available.

If yes:

- Select the alternative mechanism.
- Write a short alternative note.

Example: "Alternative: the repeated hallway check could mark suspicion rather than caution, because the action may imply distrust of someone nearby."

## What Not To Do

Do not write private internal reasoning, personal biographical details, or anything you would not want included in a de-identified research record.
