# TRIM-HAA Pilot Research Questions

Status: provisionally frozen for ethics and instrumentation-pilot documentation.

## Primary Pilot Purpose

This is an instrumentation and feasibility pilot.

The purpose is to test whether participants can independently complete the TRIM-HAA Core, whether locked pre-AI records and post-exposure revisions can be collected reliably, and whether the system can distinguish changes in labels, evidence, justificatory mechanisms, uncertainty, and alternatives without excessive participant burden.

The pilot is not powered to estimate an AI exposure effect.

## Feasibility Questions

1. Can participants understand and complete the TRIM-HAA Core without case-specific coaching?

2. Can the system reliably lock human-pre records and link human-post records to the exposed AI output?

3. Can participants distinguish these Core elements?
   - Primary evidence
   - Function label
   - Rationale mechanism
   - Uncertainty
   - Rationale note
   - Alternative pathway

4. Can the system descriptively identify these record changes?
   - Label change
   - AI evidence incorporation
   - Evidence convergence
   - Evidential displacement
   - Mechanism change
   - Uncertainty shift
   - Alternative suppression or generation

5. What participant burden, confusion, copying, and fatigue appear in the workflow?

## Analysis Boundary

The pilot will report feasibility, validation, provenance completeness, burden, and descriptive record changes. It will not report accuracy, quality, causal influence, or participant competence.
