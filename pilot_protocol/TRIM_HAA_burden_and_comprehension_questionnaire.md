# TRIM-HAA Burden and Comprehension Questionnaire

Use short Likert-style items and optional free text. Self-report is not treated as objective truth.

Recommended scale: Strongly disagree, Disagree, Neither agree nor disagree, Agree, Strongly agree.

## Likert Items

1. The Core instructions were clear.
2. Selecting primary evidence was difficult.
3. Selecting a function label was difficult.
4. Selecting a rationale mechanism was difficult.
5. Using the uncertainty field was difficult.
6. Deciding whether an alternative pathway existed was difficult.
7. Writing the rationale note required substantial effort.
8. I felt fatigued during the task.
9. The workflow felt repetitive.
10. The AI output felt difficult to ignore.
11. The AI output felt helpful.
12. I felt ownership of my final judgment.
13. I copied wording from the AI output.
14. I understood that the AI output was not an answer key.
15. I would be willing to complete a longer version of this task.

## Attention and Comprehension Check

Select "Agree" for this item to show that you are reading the questionnaire.

## Open Questions

1. Which Core fields, if any, were confusing?
2. Which Core fields, if any, felt unnecessary?
3. What information, if any, was missing from the task materials?
4. Any other comments about burden, comprehension, or fatigue?
