# v0.2.0 Usability Pilot Archive

This directory records the public archive strategy for the locked v0.2.0
ten-case usability pilot.

Raw coder submissions and completed interview/adjudication workbooks may contain
private workflow information. They are retained privately and are not committed
to the public repository. Public files use anonymous role labels only:

- Coder A
- Coder B
- Adjudicator 1

The v0.2.0 pilot remains a locked historical record. v0.2.1 revisions are
prospective and do not overwrite either coder's original annotations.

## Public Archive Contents

- `original_packet/README.md`
- `locked_submissions/README.md`
- `post_pilot_interview/README.md`
- `adjudication/README.md`
- `SHA256SUMS.txt`

## Private Retention

Private raw files are expected under a non-public project archive controlled by
the project owner. The hashes in `SHA256SUMS.txt` support verification without
publishing sensitive files.
